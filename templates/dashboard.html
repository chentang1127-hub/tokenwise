🔒 Trust & Privacy

Where is my API key stored?

Your API key lives only in your browser's localStorage. It is sent directly from your browser to the AI provider through the proxy. The TokenWise Core admin server never receives it. This is the zero-trust architecture: TokenWise Core doesn't need your key to work.

What data does TokenWise Core store?

Metadata only: model name, token counts, cost in USD, latency, and a SHA-256 hash of your prompt (used for cache deduplication).

Message content is never written to disk. The hash is a one-way fingerprint — it cannot be reversed to recover your original text.

Can anyone see my conversations?

No. Messages pass through TokenWise Core in memory only. They are forwarded to the AI provider and immediately discarded after counting tokens and computing the hash. Only the hash fingerprint is kept. The Dashboard shows model names, costs, and latency — never message text.

Where does my data live?

Everything is in a local SQLite file (tokenwise.db) in the same directory as TokenWise Core. Nothing is uploaded to the cloud or any third-party service. You can delete tokenwise.db at any time to wipe all records. TokenWise Core is fully offline-capable.
