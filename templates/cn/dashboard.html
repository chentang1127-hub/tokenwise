🔒 信任与隐私

我的 API Key 存在哪里？

您的 API Key 仅保存在浏览器的 localStorage 中。调用时由浏览器直接发送给 AI 厂商，经过代理转发。TokenWise Core 的管理服务器永远不会收到您的 Key。这是零信任架构的核心：TokenWise Core 不需要持有您的 Key 就能工作。

TokenWise Core 存储了哪些数据？

仅存储元数据：模型名、Token 数量、费用（美元）、延迟，以及您消息的 SHA-256 哈希（用于缓存去重）。

消息内容不会被写入磁盘。哈希是单向指纹 — 无法逆向还原为原始文本。

别人能看到我的对话内容吗？

不能。消息仅在 TokenWise Core 内存中临时处理 — 转发给 AI 厂商后，在完成 Token 计数和哈希计算后立即丢弃。只有哈希指纹被保留。仪表板上您可以看到：仅显示模型名、费用和延迟 — 从不显示消息文本。

数据存在哪里？

所有数据存储在 TokenWise Core 同目录下的本地 SQLite 文件（tokenwise.db）中。不会上传到云端或任何第三方服务。您可以随时删除 tokenwise.db 来清除所有记录。TokenWise Core 完全支持离线使用。
